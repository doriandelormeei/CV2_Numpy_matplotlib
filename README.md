# CV2_Numpy_matplotlib
Training on python's libraries -CV2 -Numpy -Matplotlib before to make CCTV code for RaspberryPi

Le dossier est utilisé pour se familiariser avec les éléments de manipulation d'image et de vidéo. 

Les éléments à tester doivent se situer dans le dossier ou doivent posséder un chemin d'accès si ce n'est pas le cas

Les téléchargement nécéssaire : 
-pip install opencv-python
-pip install matplotlib
-pip install numpy
